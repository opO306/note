# 인앱 결제(IAP) 상품 ID 목록

이 문서는 구글 플레이 콘솔에 등록해야 하는 인앱 상품 ID 목록입니다.

## ⚠️ 중요 사항
- 코드에 정의된 상품 ID와 구글 플레이 콘솔의 상품 ID가 **정확히 일치**해야 합니다.
- 상품 ID가 일치하지 않으면 "상품을 불러올 수 없습니다" 오류가 발생합니다.
- 배포 전에 반드시 구글 플레이 콘솔의 [인앱 상품] 메뉴에서 모든 상품이 등록되어 있는지 확인하세요.

## 상품 ID 목록

## 루멘 구매 상품
**현재 코드에는 루멘을 구매하는 IAP 상품이 정의되어 있지 않습니다.**
- 루멘은 칭호 구매에 사용되는 재화이며, 현재는 업적 달성 등을 통해 획득합니다.
- 향후 루멘 구매 기능을 추가할 경우, 이 문서에 상품 ID를 추가해야 합니다.

## 확인 방법
1. 구글 플레이 콘솔 → 앱 → 인앱 상품 메뉴로 이동
2. 위의 모든 상품 ID가 등록되어 있는지 확인
3. 상품 ID가 정확히 일치하는지 확인 (대소문자 구분)
